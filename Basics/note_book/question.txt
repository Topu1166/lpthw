Python Practice Questions:
1. Even or Odd
Write a program that asks the user to enter a number and prints whether it is even or odd.

2. Sum of Numbers from 1 to N
Write a program that takes a number n and prints the sum of all numbers from 1 to n.

3. Find the Largest Number
Given a list of numbers: [4, 9, 1, 22, 17], write code to find the largest number.

4. Check for Prime Number
Write a program that takes an integer as input and checks whether it's a prime number.

5. Reverse a String
Write a function that takes a string and returns it reversed (e.g., "hello" → "olleh").

6. Count Vowels
Write a program to count the number of vowels in a given string.

7. Fibonacci Sequence
Write a function to print the first n numbers of the Fibonacci sequence.

8. Factorial of a Number
Write a program to calculate the factorial of a given number using a for loop.

9. List of Prime Numbers (1–50)
Use a loop and logic to store all prime numbers between 1 and 50 in a list.

10. Check for Palindrome
Write a function that checks whether a word (like "madam" or "racecar") is a palindrome.
